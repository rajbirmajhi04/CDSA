Decimal:
--> Consists of only 10 digits (0 - 9)
--> Base = 10

Binary:
--> Consists of only (0 and 1)
--> Base = 2

Octal Number System: 
--> Consists only 8 digits (0 - 7)
--> Base = 8

HexaDecimal Number System: 
--> Consists only 16 alphanumeric values (0 - 15) and (A - F)
--> 0, 1, 3, 4, 5, 6, 7, 8, 9, A, B, C, D, E, F
--> A = 10, B = 11, C = 12, D = 13, E = 14, F = 15
--> Base = 16
